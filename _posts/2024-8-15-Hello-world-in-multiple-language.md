Hello, I want to teach you how to write hello world in multiple languages.
### Python
```python
print("Hello, World!")
```
### JavaScript
```javascript
console.log("Hello, World!");
```
### Rust
```rust
fn main() {
    printf("Hello, World!"); // Not sure this is correct
}
```
### C
```c
#include <stdio.h>

int main() {
    printf("Hello, World!");
}
```
